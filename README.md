# Coming Soon Landing Page – Royal Chai

## Overview
This project is a responsive "Coming Soon" landing page for a premium instant chai tea brand inspired by royal Indian households.

**Tagline:** Brew the Royal Tradition

The design reflects elegance and warmth using royal tones such as gold, deep browns, and creams.

## Features
- Clean and minimal UI
- Mobile-responsive design
- Email signup form with basic interaction
- “Launching Soon” indicator
- Social media icons
- Elegant royal-inspired color palette

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
